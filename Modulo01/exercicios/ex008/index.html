Para criar uma ordered list, vamos usar a tag <ol> para delimitar a lista e a tag <li> para identificar cada item da lista.

A tag <ol> possui um parameto type, onde configuramos o tipo de marcador da lista atual.

O marcador padrão é a bolinha preta totalmente preenchida (cicle), mas existe a opção de configurar a propriedade type da tag <ul> com os seguintes valores:

  • disc - padrão. Uma bola preta totalmente pintada
  • circle - Uma bola com uma borda e sem preenchimento
  • square - Um pequeno quadrado preto totalmente pintado
